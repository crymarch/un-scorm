# Contributing

Thank you for helping improve Un-SCORM.

## Getting started

1. Fork the repository and create a branch for your change.
2. This app is a single `index.html` (HTML, CSS, and JavaScript). Open it locally:
   - **Simple:** Open `index.html` in a browser (some features may need a local server depending on browser policies).
   - **Recommended:** Serve the folder with any static server, for example:
     - `python3 -m http.server 8080` then visit `http://localhost:8080`
     - Or the “Live Server” style extension in your editor.

## Pull requests

- Keep changes focused on one concern when possible.
- Describe what you changed and how you tested it (browser, sample SCORM type if relevant).
- Match existing code style (formatting, naming) in `index.html`.

## Issues

- Use the [bug report](https://github.com/crymarch/un-scorm/issues/new?template=bug_report.md) or [feature request](https://github.com/crymarch/un-scorm/issues/new?template=feature_request.md) templates when they apply.
- For security-sensitive reports, see [SECURITY.md](SECURITY.md).
